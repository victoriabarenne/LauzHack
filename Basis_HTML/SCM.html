Single View Mode

Welcome to Single View mode! We remind you that you can play around with the following parameters:

  • μG, the average global infections caused by a single infected individual,
  • μL, the average inter-communal infections caused by a single infected individual within this community,
  • D, the average duration of the infection, during which individuals are contagious.

Note to the user: Keep in mind that in Single Mode View, the scale of the graph adapts automatically so the graph is fitted full screen. If you want to compare graphs in a pertinent way, we suggest you use our comparison mode tab
